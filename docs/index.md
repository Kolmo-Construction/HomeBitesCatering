## Key Features

The Home Bites Catering platform includes several AI-powered features:

### 1. Lead Analysis
Analyzes vendor lead emails to extract prospect information, assess quality, and recommend actions.

### 2. Calendar Conflict Detection
Integrates with scheduling data to identify potential booking conflicts when analyzing new leads.

### 3. Sentiment Analysis
Evaluates the emotional tone of customer communications to prioritize urgent or important messages.

### 4. Content Summarization
Generates concise summaries of lengthy communications for quick review.

## Integration Points

The AI services are integrated with these system components:

- **Email System**: Processes inbound vendor leads
- **CRM**: Updates prospect and opportunity records
- **Scheduling**: Checks and flags potential calendar conflicts
- **Communication Log**: Records interaction history

## Environment Setup

The AI services require proper configuration:

1. Ensure your OpenRouter API key is configured
2. For email integration, set up the Gmail API credentials
3. Configure the target email for lead monitoring
