Source File: Primarily weddingThemeMenuData.ts for themes and packages. Other complex offerings (like "Spreads Package" from weddingHorsDoeurvesData.ts or different Sandwich Factory packages from weddingSandwichFactoryData.ts) could also be modeled as entries in menus.
Process for Each Package:
Iterate through themes and their packages in weddingThemeMenuData.ts.
For each package, prepare a record for the menus table:
name: Package name (e.g., "Bronze Celebration Package").
description: Package description.
type: Theme key (e.g., "taste_of_italy_wedding").
items (JSONB): Construct the JSON object based on the structure you finalized in Step 1.
This JSON will include theme_key, package_price_per_person, min_guest_count.
The categories array within this JSON will list the categories relevant to this package.
For each category, available_item_ids will be an array of integer IDs obtained from your idMap (created while populating menu_items).
selection_limit will come from package.limits.
Insert into menus table.
