Test Case CI-1: Create Contact Identifier for a Lead

Endpoint: POST /api/contact-identifiers
Method: POST
Headers: Content-Type: application/json, (ensure session cookie is sent if needed for auth)
Body:
JSON

{
  "leadId": 1, // Replace with an actual leadId
  "type": "email",
  "value": "testlead@example.com",
  "isPrimary": true,
  "source": "manual_test"
}
Expected Status: 201 Created
Expected Response Body: JSON object of the created identifier, including an id. Note this id for later tests.
Verification: Check your contact_identifiers table in the database.

Endpoint: POST /api/contact-identifiers
Method: POST
Body:
JSON

{
  "type": "email",
  "value": "nocontact@example.com",
  "source": "manual_test"
}
